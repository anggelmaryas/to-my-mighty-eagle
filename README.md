# to-my-mighty-eagle
A small birthday website with a hidden message challenge, created with love as a digital gift &lt;3

A Capture The Flag (CTF)-styled digital love letter designed as a birthday surprise.
This project combines cyber security concepts, gamification, and emotional storytelling into an interactive web-based experience.

The recipient must complete each challenge to unlock heartfelt messages, making the journey both fun and meaningful.

Project Concept

Cyber Love CTF transforms a birthday letter into a series of missions:

Each stage represents a CTF challenge

Correct answers unlock personal messages

Progression is linear, creating suspense and emotional rewards

This project is intentionally built without external libraries to keep it simple, lightweight, and easily shareable.
